   → python Interface.py

🧪 "Je veux tester d'abord que ça marche"
   → python test_interface.py
   └─ Exécute 2 tests complets (minimal + maison)

📖 "Je veux lire la documentation"
   → RESUME.txt (vue générale rapide)
   → README_GUI.md (documentation complète)
   → GUIDE_COMPLET.py (voir avec: python GUIDE_COMPLET.py)

   → Ouvrir: config_exemple.py
   └─ Contient configurations prédéfinies

   → Ouvrir: Interface.py
   └─ Classe SolaireGUI (ligne 31)

📋 "Je veux comprendre les calculs"
   → Ouvrir: Services/ConsommationService.py
   └─ Fonction: dimensionnerSystemeSolaire()

1. ✅ Interface.py (1000+ lignes)
   Type: Script principal (Classe + fonction main)
   Description: Interface graphique complète avec tkinter
   Usage: python Interface.py
   Contient:
     • Classe SolaireGUI (interface GUI)

   Type: Script wrapper de lancement
   Description: Démarrage + affichage instructions
   Usage: python run_interface.py (alternatif à Interface.py)
   Contient:
     • Instructions d'utilisation
     • Affichage dans le terminal
     • Appel à SolaireGUI

3. ✅ test_interface.py (300+ lignes)
   Type: Script de test
   Description: Tests fonctionnels avec exemples
   Usage: python test_interface.py
   Contient:
     • Test 1: Configuration MINIMALE
     • Test 2: Configuration MAISON
     • Affiche résultats détaillés
   Utilité: Vérifie que tout fonctionne avant utilisation

4. ✅ config_exemple.py (300+ lignes)
   Type: Module de configuration
   Description: Données prédéfinies et exemples
   Usage: from config_exemple import ...
   Contient:
     • CONFIG_JOURNEE (3 périodes)
     • CONFIG_BATTERIE (batterie)
     • APPAREILS_EXEMPLE_* (exemple minimal/maison/complet)
     • PROFILS_SOLAIRES (tropical/tempéré/montagne)
     • CAPACITES_BATTERIE_RECOMMANDEES
   Utilité: Référence + données de test

JOUR 1 - DÉCOUVERTE:
  1. Lire RESUME.txt (10 min)
