⚡ 3 CONCEPTS CLÉS À RETENIR
═════════════════════════════════════════════════════════════════════════

1. PUISSANCE PRATIQUE
   → Énergie que vous utilisez/produisez réellement
   → Tient compte des rendements et besoins
   → Exemple: 500W pour la batterie

2. PUISSANCE THÉORIQUE
   → Spécification du MATÉRIEL À ACHETER
   → Inclut marge de sécurité (+ 50% batterie)
   → Ou inverse du rendement (÷ 40% pour panneau)
   → Exemple: 750W à acheter pour batterie 500W pratique

3. RENDEMENT
   → % d'efficacité (moins le soleil est haut, moins bon)
   → Matin: 40% (soleil bas) → Panneau moins efficace
   → Après-midi: 100% (soleil haut) → Rendement max
   → Soir: 0% (pas de soleil) → Pas de production

🔧 VÉRIFICATION PRÉREQUIS
═════════════════════════════════════════════════════════════════════════

Vous avez besoin de Python 3.8+ avec tkinter.

VÉRIFIER:
  $ python3 --version
  $ python3 -c "import tkinter; print('OK')"

Si tkinter manque, installez:
  
  Linux (Ubuntu/Debian):
    $ sudo apt-get install python3-tk
  
  Linux (Fedora):
    $ sudo dnf install python3-tkinter
  
  macOS:
    $ brew install python-tk
  
  Windows:
    Réinstaller Python en cochant "tcl/tk"

Q: "Ça marche vraiment?"
R: Oui! Exécutez test_interface.py pour voir les tests passer.

Q: "Je dois entrer quoi comme consommation?"
R: Watts (W). Exemples: Réfrigérateur 150W, PC 200W, Lumière 40W

Q: "Quel format d'heure?"
R: HH:MM:SS exactement. "06:00:00" pas "6:00:00"

Q: "Je dois acheter quoi?"
R: Les "Puissance théorique"! 
   - Batterie: XXX Wh
   - Panneau: XXX W

Q: "Différence pratique vs théorique?"
R: Pratique = ce que vous utilisez
   Théorique = avec marge, ce que vous achetez
